# Nexus Elements

Nexus Elements is a premium collection of custom Elementor widgets.

## Features

- **Interactive Switcher**: A dynamic widget for creating interactive toggle content.
- **Nexus Gallery**: A beautifully designed gallery widget for Elementor.
- **Nexus Video Library Integration**: Seamlessly integrates with the Nexus Video Library if the addon is active.

## Requirements

- WordPress
- Elementor 3.0.0 or greater

## Installation

1. Download the plugin as a zip file.
2. Go to **Plugins > Add New** in your WordPress dashboard.
3. Click on **Upload Plugin** and select the zip file.
4. Activate the plugin after successful installation.
5. Ensure Elementor is installed and activated.
